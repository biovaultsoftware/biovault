BioVault PWA â€” Decentralized MSL (Micro Segments Ledger)

BioVault is a decentralized, offline-first Progressive Web App implementing the BalanceChain MSL protocol. It enables peer-to-peer value transfer without intermediaries, supports offline operation, and allows users to manage SHE-denominated segments, generate cryptographic proofs, and perform secure P2P transfers. An optional on-chain TVM gateway is available for settlement, swaps, and interoperability with blockchain systems.

BalanceChain White Paper â€” Abstract

BalanceChain is a human-centric micro-ledger protocol designed to support secure, auditable value transfer without reliance on global consensus networks. The system is grounded in Standard Human Effort (SHE) as a unit of economic reference, with the Efficiency Correction Factor (ECF) providing a contextual adjustment layer for comparative economic analysis.

The protocol defines True Value of Money (TVM) as an on-chain settlement and interoperability instrument, with a fixed internal conversion ratio of 1 TVM = 12 SHE. TVM is not a sovereign currency and does not represent a price peg to any fiat or digital asset.

BalanceChain operates as a hybrid off-chain / on-chain system. Users maintain a personal, tamper-resistant micro-ledger locally on their device, secured through modern cryptography and biometric authentication. Time-ordered segment proofs enable deterministic validation, offline operation, and secure ownership transfer, while optional on-chain interactions allow external settlement without exposing full transaction histories.

Centralized SCL (Serial Centric Ledger) Overview

For central banks, banks, money transfer companies, and centralized entities. Enables CBDC-fiat twins with full traceability, instant/offline zero-fee transfers for any currency (e.g., USD, Euro, Riyal). This PWA provides a demo/simulation; full implementation requires server-side integration.

SCL Key Features

  • Serial-Centric Ledger: Tracks every unit (cash/digital) by serial, denomination, events, and ownership chain.
  • CBDC-Fiat Twins: Seamless switch between physical cash and digital, fully synced and auditable for any currency.
  • Full Traceability: Instant tracing, anti-laundering, anti-theft, anti-robbery. Flags stolen/forced transfers in real-time.
  • Regulator-Owned: No blockchain risks, central control with offline capabilities and zero fees.
  • Integration: Software upgrades for ATMs/banks; no disruption to users.
  • Superiority: Unlimited scalability, proactive crime prevention, meets global AML standards.

TVM On-Chain Gateway (Distributed Ledger Interface)

Connect a compatible wallet to interact with the TVM smart contract for claims, exchanges, and settlement operations. The gateway applies a fixed internal conversion of 12 SHE per TVM and supports protocol-defined dynamic valuation.

This interface serves as the on-chain interoperability layer for both MSL (decentralized) and SCL (centralized) BalanceChain systems. Actions are automatically populated from verified BalanceChain proofs to ensure a streamlined and consistent user experience.
